# Moni Front

## Instalación

```bash
# cambio de directorio
cd bitcoin-web

# instalar dependencias
npm install

# Servir en localhost: 3000
npm run dev

# Para comenzar la versión de compilación
# construir para producción
npm run build

# iniciar el servidor de compilación
npm run start

# Ahora puede encontrar la carpeta de compilación "./next" en el directorio principal, puede usarla para implementar en el servidor de producción

```
